You are pointing at a real gap.

The shallow explanation says:

The RNN hidden state “summarizes the previous tokens,” then the model “uses context,” then backprop “learns the right representation.”

That is not a mechanistic explanation. It is mostly a behavioral description.

The stricter version is:

ht=ϕ(Wxxt+Whht−1+b)h_t = \phi(W_x x_t + W_h h_{t-1} + b)ht​=ϕ(Wx​xt​+Wh​ht−1​+b)

This equation does not say the hidden state “understands,” “summarizes,” or “contains meaning.” It only says:

the hidden state is a recursively updated vector whose values are chosen so that the final input-output behavior reduces loss.

That is much weaker.

A better way to say it:

The hidden state is not intrinsically a “context vector.” It is a learned internal state of a dynamical system. Its only formal meaning is its causal role in producing future outputs.

The missing thing is usually this:

There is no guaranteed semantic mapping from human concepts to hidden-state coordinates.

People often talk as if:

hidden state = compressed meaning of the past

But the actual thing is closer to:

hidden state = whatever internal numerical configuration helps the model approximate the training objective

Those are not the same.

A hidden state may encode tense, topic, entity identity, syntax, or long-range dependency. But it does not have to encode them in a clean, human-readable, symbol-like way. It can encode them in an entangled, distributed, non-identifiable way. Even worse, different trained models can implement similar behavior with different internal geometries.

So when someone says:

“The hidden state summarizes the previous sequence.”

The skeptical correction is:

It behaves as if it preserves some task-relevant information from the previous sequence, but we have not specified what information, in what basis, with what invariants, or by what mechanism it becomes semantically interpretable.

That is the gap.

Backpropagation also does not magically create “understanding.” It searches parameter space for weights that reduce a scalar objective. The loss function constrains outputs, not necessarily internal meanings. So the model can learn useful internal structure, but the training objective does not directly say:

make neuron 17 represent subject plurality
make dimension 42 represent the current topic
make this subspace represent speaker intent

Those structures may emerge, but emergence is not explanation.

A more rigorous interpretation would be:

For a given task, the hidden state should preserve distinctions between histories that matter for future predictions and collapse distinctions that do not matter.

That is much closer to the real logic.

For example, in language modeling, two different prefixes may be equivalent if they imply the same distribution over likely next tokens. The hidden state does not need to “summarize everything.” It only needs to preserve enough information to make useful future predictions.

So the true object is not “meaning” in the human sense. It is more like a learned predictive state:

history → internal state → prediction

The hidden state’s “meaning” is relational and functional, not intrinsic.

Your criticism is valid when people imply that words like “embedding,” “context,” “representation,” “attention,” or “hidden state” explain the mechanism. They often do not. They name components. They do not explain why the learned geometry has the properties people attribute to it.

The clean distinction is:

Mathematical object:
    a vector updated by a recurrence.

Optimization role:
    a variable adjusted indirectly through loss minimization.

Functional role:
    a compressed state used to predict outputs.

Semantic interpretation:
    optional, post-hoc, non-unique, and often only partially true.

So the statement “RNN hidden states summarize context” is acceptable only as loose shorthand. The precise statement is:

An RNN hidden state is a learned, non-canonical, task-dependent internal state that may encode information about previous inputs insofar as that information helps reduce the training loss.

That is much less magical, and much more honest.
